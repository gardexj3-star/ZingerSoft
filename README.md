# ZingerSoft Client

Клиент ZingerSoft для Minecraft 1.12.2.

## Возможности

- **50+ модулей** различных категорий (Combat, Movement, Player, Render, World, Misc)
- **Система команд** с настраиваемым префиксом
- **ClickGUI** для удобного управления модулями
- **HUD** с отображением активных модулей
- **Система конфигураций** для сохранения настроек

## Сборка

См. файл `build.txt` для подробных инструкций.

Кратко:
```bash
mvn clean package
```

Готовый JAR будет в `target/zingersoft-1.0.0.jar`

## Установка

1. Установите Minecraft Forge 1.12.2 (версия 14.23.5.2860)
2. Скопируйте JAR в папку `mods`
3. Запустите Minecraft

## Использование

- **Right Shift** - открыть ClickGUI
- **.** - префикс команд (можно изменить через `.prefix`)
- **Команды:**
  - `.help` - список команд
  - `.toggle <module>` - включить/выключить модуль
  - `.bind <module> <key>` - привязать клавишу
  - `.config save/load <name>` - управление конфигами

## Структура проекта

```
src/main/java/com/zingersoft/
├── ZingerSoft.java          # Главный класс мода
├── module/                   # Система модулей
│   ├── Module.java          # Базовый класс модуля
│   ├── ModuleManager.java   # Менеджер модулей
│   ├── combat/              # Боевые модули
│   ├── movement/            # Модули движения
│   ├── player/              # Игровые модули
│   ├── render/              # Визуальные модули
│   ├── world/               # Модули мира
│   └── misc/                # Разное
├── command/                 # Система команд
├── gui/                     # Интерфейсы
└── config/                  # Конфигурации
```
